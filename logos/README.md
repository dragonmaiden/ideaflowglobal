# Client Logos

Please add the following logo files to this directory:

1. `nice-investors.png` - NICE Investors logo
2. `gobiz-korea.png` - Gobiz Korea logo
3. `kosme.png` - KOSME logo
4. `kyobo.png` - Kyobo logo

**Recommended specifications:**
- Format: PNG with transparent background
- Max height: 48px (logos will scale proportionally)
- Max width: 180px
- Color: Preferably in brand colors (will be grayscale by default, color on hover)

If logo files are not found, the brand names will be displayed as text with larger font.
